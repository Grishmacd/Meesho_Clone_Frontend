# Myntra Clone (Frontend)

A Myntra-style homepage UI clone built using only HTML and CSS. This project recreates the look and layout of a typical Myntra landing page, including the navigation header, banners, product highlight sections, shop-by-category grid, and a detailed footer. Built to practice real-world UI layout and CSS positioning without frameworks.

Disclaimer: This project is for educational/portfolio use only. Myntra branding and assets belong to their respective owners. This project is not affiliated with Myntra.


## Project Overview

This project recreates a Myntra-like homepage interface, including:
- Header with Myntra logo and category navigation (Men, Women, Kids, Home, Beauty, etc.)
- Search bar section and action icons (profile, wishlist, bag)
- Promotional coupon and main banner sections
- Product highlight sections like Rising Stars and brand deal rows
- Shop By Category image grid section
- Footer similar to Myntra layout (shopping links, policies, app badges, social icons, office address)

This is a frontend-only clone created to improve HTML structure and CSS styling skills.

## How to Run Locally

### Option 1: Open Directly
1. Download or clone this repository.
2. Open `index.html` in your browser.

### Option 2: VS Code Live Server (Recommended)
1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html` → **Open with Live Server**.
