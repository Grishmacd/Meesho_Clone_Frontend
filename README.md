# Meesho Clone (Frontend)

This project is a Meesho-style homepage UI clone made using HTML and CSS. It recreates the look and layout of a Meesho shopping homepage with a header + search, category navigation, hero/banner sections, product option cards, a product listing grid, and a footer.
It is built only for learning, practice, and portfolio use.

## What this project is
A static frontend clone (UI) that focuses on:
- Clean e-commerce UI styling similar to Meesho
- Section-based layout (header, categories, banners, product grid, footer)
- Practicing CSS layout, spacing, and alignment (no frameworks)

## What you will see in the website
- Header with logo, search bar, and top navigation items
- Category navigation bar (shopping sections)
- Hero/banner sections and promotional strips
- Product option/category cards section
- Product listing layout (grid-style cards)
- Footer with multiple links and app/social section

## Features
- Top navbar with search + navigation
- Category menu layout similar to shopping sites
- Banner/promotional sections
- Product/category cards
- Product grid layout for listings
- Structured footer section

## Tech Stack
- HTML5
- CSS3
